# Waste Detection + Tracking

## Description
This project uses YOLOv5 for waste detection and ByteTrack for tracking.

## Files
- tracking.py → main code
- best.pt → trained model (shared separately)
- input.mp4 → sample input
- output.mp4 → output result

## How to Run
1. Install:
   pip install torch opencv-python supervision ultralytics

2. Run:
   python tracking.py
